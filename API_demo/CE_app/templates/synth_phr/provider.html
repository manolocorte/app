Provider Help Menu

This endpoint returns full details on a Medicare-listed, US-based healthcare provider.

DRAFT

Available Options

Endpoint Returns
/api/v01/provider?entity_type=nnn Returns random provider within the selected entity type.
Options:
"1" for an individual person
"2" for an organization
&number=nnn This parameter defines the number of records to return.
Anonymous (free) users can request up to 05 records at a time.
Paid subscribers can request up to 100 records at a time.
(Default = 1 record).
/api/v01/provider?entity_type=nnn&state=[character] Returns random provider within the selected state(s) and entity type.
/api/v01/provider?entity_type=nnn&city=[character] Returns random provider within the selected city(ies) and entity type.
/api/v01/provider?npi=nnn Returns random provider within the selected npi number.
/api/v01/provider?help_states Returns a list of codes for available states.
/api/v01/provider?state=[character]&help_cities Returns a list of codes for available cities within the selected state.
/api/v01/provider?help This help menu.
