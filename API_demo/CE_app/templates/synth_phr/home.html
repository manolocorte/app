Functional Blocks

These are the functional blocks the NIHPO Synthetic People Platform implements.

Each Functional Block is exposed as a fully-formed API.

Endpoint Returns
Condition A scientifically-valid health condition.
Using concepts from the SNOMED CT hierarchy "Clinical finding".
Device A medical device approved for use in the US.
Drug Full details of a drug approved for use in either the EU or the US.
Geography A location in a province within a country anywhere in the world.
Lab Result A scientifically-valid laboratory result set.
Using the Universal Lab Orders Value Set from LOINC. These LOINC codes cover more than 95% of lab test orders in the U.S.
Procedure Full details of a medical procedure.
Using a list of the most commonly-used procedures in SNOMED-CT.
Provider Full details on a Medicare-listed, US-based healthcare provider.
Vitals A scientifically-valid human body vital sign reading.
