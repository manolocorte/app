Drug Help Menu

This endpoint returns full details of a drug approved for use in either the EU or the US.

DRAFT

Available Options For FDA (Food and Drug Administration) Records

Endpoint Returns
/api/v01/drug?regulator=us_fda Returns random drug(s) approved by the FDA.
&number=nnn This parameter defines the number of records to return.
Anonymous (free) users can request up to 05 records at a time.
Paid subscribers can request up to 100 records at a time.
(Default = 1 record).
/api/v01/drug?regulator=us_fda&active_ingredient=abc Returns random drug(s) within the selected active ingredient.
/api/v01/drug?regulator=us_fda&appno=nnn Returns a full record for an individual Application Number.
/api/v01/drug?regulator=us_fda&form=abc Returns random drug(s) within the selected form of administration.
Options: "AEROSOL"; "BAR"; "CAPSULE"; "CLOTH"; "CONCENTRATE"; "CREAM"; "DISC"; "EMULSION"; "ENEMA"; "FIBER"; "FILM"; "FOAM"; "GAS"; "GEL"; "GRANULE"; "GUM"; "IMPLANT"; "INHALANT"; "INJECTABLE"; "LIQUID"; "OINTMENT"; "PASTE"; "PASTILLE"; "PATCH"; "PELLET"; "POWDER"; "SHAMPOO"; "SOAP"; "SOLUTION"; "SPONGE"; "SPRAY"; "SUPPOSITORY"; "SUSPENSION"; "SWAB"; "SYRUP"; "TABLET"; "VIAL"
/api/v01/drug?regulator=us_fda&marketing_status=abc Returns random drug(s) within the selected marketing status.
Options for "us_fda" regulator:
"Disc" for "Discontinued"
"None" for "None (Tentative Approval)"
"Otc" for "Over-the-counter"
"Pre" for "Prescription"
/api/v01/drug?regulator=us_fda&name=abc Returns random drug(s) within the selected name.
/api/v01/drug?regulator=us_fda&sponsor=abc Returns random drug(s) within the selected sponsor.
/api/v01/drug?regulator=us_fda&submission_type=abc Returns random drug(s) within the selected submission type.
Options:
"ORIG"
"SUPPL"
/api/v01/drug?regulator=us_fda&help_active_ingredients=[character] Returns a list of all active ingredients that start with the single character.
/api/v01/drug?regulator=us_fda&help_names=[character] Returns a list of all drug names that start with the single character.
/api/v01/drug?regulator=us_fda&help_sponsors=[character] Returns a list of all sponsors that start with the single character.
/api/v01/drug?help This help menu.
