Condition Help Menu

This endpoint returns a scientifically-valid health condition.
Using concepts from the SNOMED CT hierarchy "Clinical finding".

Available Options

Endpoint Returns
/api/v01/condition Returns random condition(s).
&number=nnn This parameter defines the number of records to return.
Anonymous (free) users can request up to 05 records at a time.
Paid subscribers can request up to 100 records at a time.
(Default = 1 record).
/api/v01/condition?record_type=[hierarchy_list] Types of concepts.
Options: "disorder"; "finding"; "situation"
The following options, while present in the dataset, are not made available due to the low number of values:
"event": "63"
"morphologic abnormality": "46"
"navigational concept": "2"
"person": "2"
/api/v01/condition?source=snomed_ct_core Selected information source.
Options: "snomed_ct_core"
Currently we only have the Clinical Observations Recordings and Encoding (CORE) Problem List Subset of SNOMED CT available.
