Geography Help Menu

This endpoint returns a location in a province within a country anywhere in the world.

DRAFT

Available Options

Endpoint Returns
/api/v01/geography Returns random geographical location(s).
&number=nnn This parameter defines the number of records to return.
Anonymous (free) users can request up to 05 records at a time.
Paid subscribers can request up to 100 records at a time.
(Default = 1 record).
&randomize=0 The parameter “randomize=0” will turn off the slight randomization of Lat/ Long of each returned location.
This randomization will return a Lat / Long that is slightly off the default Lat / Long for the returned location.
This randomization helps avoid multiple points overlapping on a single Lat/Long point when displaying the returned location on a map.
(Default = 1, do randomize locations)..
/api/v01/geography?country=[country_code_list] Full random geographical location within the selected country(ies).
(Default = 1 record).
/api/v01/geography?country=[country_code]&state=[state_name_list] Full random geographical location within the selected country, and state(s).
/api/v01/geography?help_countries List of codes for available countries.
/api/v01/geography?help_country=[country_code]&states List of states / provinces for the selected country.
/api/v01/geography?help_country=[country_code]&state=[state_code]&locations List of locations in this state / province within the defined country.
/api/v01/geography?help This help menu.
