Lab Result Help Menu

This endpoint returns a scientifically-valid laboratory result set. Using the Universal Lab Orders Value Set from LOINC. These LOINC codes cover more than 95% of lab test orders in the U.S.

DRAFT

Available Options

Endpoint Returns
/api/v01/lab_result Returns random lab result(s).
&number=nnn This parameter defines the number of records to return.
Anonymous (free) users can request up to 05 records at a time.
Paid subscribers can request up to 100 records at a time.
(Default = 1 record).
/api/v01/lab_result?cdisc_common_tests=[y/n] Returns random lab result(s) within terms that are part of a subset of LOINC terms used by CDISC clinical trials.
Options:
"y"
/api/v01/lab_result?class_type=[classtype_list] Returns random lab result(s) within the selected class type(s).
Options:
"1" for Laboratory class
"2" for Clinical class
"3" for Claims attachments
"4" for Surveys
/api/v01/lab_result?loinc_num=[LOINC number] Returns full lab result for the selected LOINC number.
/api/v01/lab_result?panel_type=[panel_type_list] Returns random lab result(s) within the selected panel type(s).
Options:
"convenience"
"organizer"
"panel"
/api/v01/lab_result?rank=[rank_list] Returns random lab result(s) within the selected rank(s).
Options:
"order" for COMMON_ORDER_RANK {Ranking of approximately 300 common orders performed by laboratories in USA.}
"si_test" for COMMON_SI_TEST_RANK {Corresponding SI terms for 2,000 common tests performed by laboratories in USA.}
"test" for COMMON_TEST_RANK {Ranking of approximately 2,000 common tests performed by laboratories in USA.}
/api/v01/lab_result?status=[status_list] Returns random lab result(s) within the selected status(es).
Options:
"deprecated"
"discouraged"
"trial"
"Active" terms will be displayed by default. User must specify additional status codes.
/api/v01/lab_result?source=[source_list] Returns random lab result(s) within the selected source(s).
Options:
"loinc"
"discouraged"
Currently we only have the Logical Observation Identifiers Names and Codes (LOINC) data available.
/api/v01/lab_result?help This help menu.
